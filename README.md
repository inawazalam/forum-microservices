# Traceable GoLang application for forun module


## Requirements
- [GoLang](https://golang.org/doc/install)
- [Mongo DB](https://docs.mongodb.com/manual/installation/)
- [Postgres DB](https://www.postgresql.org/docs/9.3/tutorial-install.html)
- [Traceable - Spring Boot - User microservices application](https://git.hashedin.com/nawaz.alam/traceable-user-microservices)
[Traceable - React Application- User microservices application](https://git.hashedin.com/nishanth.p/traceable-web)
- [Traceable - Python Django - Merchant microservices application](https://git.hashedin.com/nishanth.p/traceable-merchant)

#### Steps to Build Application:

- On the command line
```shell
go build
```

#### Steps to pull and install dependencies for application:

```shell
git clone https://github.com/inawazalam/forum-microservices
cd forum-microservices
go build

```

#### Steps to run the application:

```shell
go run main.go
```
Runs the app in the development mode.  
Open [http://localhost:8088](http://localhost:8088) to view it in the browser.
